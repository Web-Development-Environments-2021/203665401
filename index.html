About Me

.שמי אלי (אליהו ) שטרית

.נולדתי בשנת 1925 בעיירה ארפוד שבדרום מרוקו, בן בכור לאברהם ופרחה שטרית (ז"ל) ואח לאסתר(ז"ל)

.בילדותי למדתי בבית הספר היסודי בעיירה ובמקביל למדתי בבית התלמוד של הבבא סאלי בעיקר לימודי תורה. תוך כדי, למדתי את מקצוע הצורפות, כדי להמשיך את מלאכת אבי

.בשנת 1947 נשאתי לאישה את עדה נזרי, נערה יפיפיה ואצילית, הצעירה ממני בשבע שנים

.כאשר שרלי, בני הבכור היה בן שנה, החלטתי לנסות שוב את מזלי בעיר הגדולה אורון, והפעם עם כל בני משפחתי, בני בכורי והורי. לפרנסתי רכשתי משאית ועסקתי במסחר של ממתקים

.באלגיריה נולדו לנו עוד 5 ילדים (דוד , ז'וסלין שושנה ז"ל, שמעון, מוריס, ג'רר), מצבנו הכלכלי השתפר מאוד . גרנו באורון כ18 שנים הילדים למדו בבית ספר יהודי "אליאנס", ביום ראשון נהגנו ללכת עם הילדים לקולנוע, לחוף הים ולבתי קפה. עד שפרצה מלחמת האזרחים באלג'יריה

.הגענו לארץ בשנת 1961 באוניה ממרסיי לחיפה. לא שאלו אותנו יותר מדי שאלות וכך הגענו לאשקלון במשאית כל המשפחה: ההורים שלי, אשתי ואני וששת ילדנו

.התקשיתי להתאקלם בארץ. לכולם היה קשה. קניתי משאית והתחלתי להוביל עליה סחורות... לא הייתה עבודה... והכסף שהבאנו איתנו הלך ונגמר

.חזרתי לצרפת, למרסיי. עבדתי כנהג אוטובוס. גרתי שם שנתיים אצל בני משפחה. אספתי כסף, קניתי בית למשפחה וחזרתי לארץ כדי להביא אותם לצרפת

.אז נשארתי בארץ. התקבלתי לעבודה במועצה הדתית של אשקלון. ונולדו לנו עוד שתי בנות מרטין וציפי

.תחביבים שלי הם קריאה הליכה רגלית לחוף הים

.מוסיקה אהובה עלי היא קסידות- בלאדות מרוקאיות. שייח' מואיז'ו, פריד אל-אטרש

.מנהג אחד אני נוהג עד היום כאבי: ביום שבת, לאחר סעודת השבת, בעודנו מסבים לשולחן , אני מספר לילדים ולנכדים סיפור או מעשייה

Q&A

Question 1

Three main diffrences between Web 1.0 , Web 2.0 , Web 3.0:

  1. Web 1.0 is mostly read-only and static, not dynamic. Web 2.0 is wildly read-write and Web 3.0 is portable and personal. it means that in Web 2.0 and Web 3.0 allow the user to edit and share contect.
  2. in Web 1.0 there was no search engine so the user must know the URL to approach the website. in Web 2.0 the user could search on search engine like google to get access to content related to the query that the user searched for, but not answer to specific question for example. and in Web 3.0 there is AI algorithems that can return an exact answer to the query the user searched for.
  3. Web 1.0 uses banner advertising, Web 2.0 uses Interactive advertising and Web 3.0 uses behavioural advertising.




Question 2

The diffrences in a Use Case: Pizza ordering

  • Web 1.0: if the user want to order Pizza from "Pizza Plus" - he must know the URL.
  • Web 2.0: the user will write in the search engine "Pizza Plus" and the results will be pages with tags of the query "Pizza Plus".
  • Web 3.0: the user will write in the search engine "Pizza Plus" and the results will be spesific website of the query.
