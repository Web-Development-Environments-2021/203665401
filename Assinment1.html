Question 1

Three main diffrences between Web 1.0 , Web 2.0 , Web 3.0:

  1. Web 1.0 is mostly read-only and static, not dynamic. Web 2.0 is wildly read-write and Web 3.0 is portable and personal. it means that in Web 2.0 and Web 3.0 allow the user to edit and share contect.
  2. in Web 1.0 there was no search engine so the user must know the URL to approach the website. in Web 2.0 the user could search on search engine like google to get access to content related to the query that the user searched for, but not answer to specific question for example. and in Web 3.0 there is AI algorithems that can return an exact answer to the query the user searched for.
  3. Web 1.0 uses banner advertising, Web 2.0 uses Interactive advertising and Web 3.0 uses behavioural advertising.




Question 2

The diffrences in a Use Case: Pizza ordering

  • Web 1.0: if the user want to order Pizza from "Pizza Plus" - he must know the URL.
  • Web 2.0: the user will write in the search engine "Pizza Plus" and the results will be pages with tags of the query "Pizza Plus".
  • Web 3.0: the user will write in the search engine "Pizza Plus" and the results will be spesific website of the query.
